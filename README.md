# GGMNIRA Reproducible Materials

This repository contains the reproducible materials for the empirical application of the Gaussian Graphical Model NodeIdentifyR Algorithm (GGMNIRA).

The materials provide a step by step R Markdown example showing how to apply GGMNIRA to the built-in `RIdata` data set from the `GGMNIRA` R package. The example also includes a comparison with the original Ising-model-based NIRA framework after recoding the ordinal data into binary variables.

## Contents

* `GGMNIRA_tutorial.Rmd`
  The R Markdown source file containing the full reproducible analysis.

* `GGMNIRA_tutorial.html`
  The rendered HTML file containing the code, results, figures, and explanatory text.

* `GGMNIRA_tutorial_files/`
  Supporting files generated by R Markdown, such as figures and HTML dependencies.

## Analysis overview

The tutorial includes the following steps:

1. load the `GGMNIRA` R package and the built-in `RIdata` data set;
2. estimate a regularized Gaussian graphical model for the Rest Intolerance items;
3. apply GGMNIRA to compute KL divergence values after simulated manipulation of each node;
4. visualize the projected manipulation effects;
5. evaluate the stability of the KL divergence estimates using a case-dropping bootstrap;
6. conduct bootstrap difference tests for pairwise differences in KL divergence;
7. recode the original ordinal data into binary;
8. estimate a regularized Ising model for the binary Rest Intolerance items;
9. apply the original NIRA to the Ising model under alleviating and aggravating manipulations.

## Installation

The `GGMNIRA` package can be installed from GitHub using:

```r
install.packages("remotes")
remotes::install_github("ywu6888/GGMNIRA")
```

To perform the original NIRA analysis based on the binary Ising model, the following additional packages are also required.

```{r installation-original-nira, eval=FALSE}
install.packages(c("nodeIdentifyR", "dplyr"))
```

## How to reproduce the analysis

To reproduce the analysis locally:

1. Download or clone this repository.
2. Open `GGMNIRA_tutorial.Rmd` in RStudio.
3. Install the required packages.
4. Click `Knit` to render the R Markdown document.
5. The analysis will generate an HTML file containing the code, results, figures, and explanatory text.
